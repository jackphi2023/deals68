# Homepage H0 baseline

## Purpose

This directory records the visual and source baseline for the Deals68 Homepage before the H1–H5 CSS stabilization series.

H0 is evidence-only. It must not change Homepage runtime UI, data loading, routes, Supabase, banner behaviour, CSS imports, Netlify, or production configuration.

- Repository: `jackphi2023/deals68`

- Netlify must not be deployed by H0.

## Known baseline findings

H0 intentionally records the current cascade, including known cleanup targets:

1. `home.css` contains the earlier mobile Hero `contain` contract.
2. The later imported `home-hero-media.css` enforces `cover !important` at runtime.
3. `release-foundation.css` adds `50px !important` vertical padding to Home Promotion.
4. Homepage selectors still exist in `ui-fixes.css` and `release-foundation.css`.
5. Featured Investor cards are defined in more than one stylesheet.
6. `release-cleanup.css` is a comment-only compatibility stub.

These findings are evidence for H1–H5. H0 does not fix them.
